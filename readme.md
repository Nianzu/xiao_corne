# Toolchain
```
sudo apt install build-essential
cargo install espup --locked 
espup install
cat $HOME/export-esp.sh >> ~/.bashrc
```
# Tooling
```
cargo install esp-generate --locked
cargo install espflash --locked
cargo install esp-config --features=tui --locked
```
